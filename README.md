# Advanced System to implement Congestion Control and Traffic Regulations

The solution aims to build a single system with combined features achieving below goals:

  •	Seamlessly Monitoring Vehicle Presence at Traffic Signals <br/>
  •	Alerting Drivers Upon Exceeding Speed Limits Using Vehicle Speed Sensors <br/>
  •	Utilizing Ultrasonic Sensors for Real-time Traffic Monitoring and Classification <br/>

# Technologies and Components <br/>
•	Arduino

•	Embedded Programming

•	IoT

•	NodeMCU

•	RFID

•	LED

•	LM393 Speed Sensor

•	Ultrasonic Sensor

•	Thingsboard


# Product Features <br/>
### Intelligent RFID Detection <br/>
### Enhanced Speed Monitoring <br/>
### Traffic Density Analysis <br/>
